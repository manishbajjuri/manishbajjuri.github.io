IRIS is data which is available on sklearn.
It is classification based problem.


Features :

1. Sepal length
  DataType : Float and Data Range 3.5 - 8.0
2. Sepal width
  DataType : Float and Data Range 2.0 - 5.0
3. Petal length
  DataType : Float and Data Range 1.0 - 7.0
4. Petal width
  DataType : Float and Data Range 0.1 - 2.8

Target Class:

1 - Iris Setosa
  Class : 0
2 - Iris Versicolour
  Class : 1
3 - Iris Virginica
  Class : 2

Description

Number of record's in the data is 150.
This 150 records are spited into train data and test data by 70:30 ratio.
Then 105 record's will be train data and remaining 45 are test data.
As the Size of the Data set is very less(150 records). Applying Cross Validation can make the model to perform better.
This data is trained on different Machine learning classifier algorithm.
Performance of all algorithms are listed below.

Algorithm Accuracy Cross Validation
1 Logistic Regression 96.6 97.3
2 Decision Trees 96.6 95.30
3 Random Forest 96.6 96.5
4 AdaBoost 96.6 95.34
5 Gradient Boost 98.3 96.0
6 XGBoost 96.6 96.0


By observing above accuracy table we can easily conclude that Logistic Regression is performing better result with 97.3 accuracy compare to all different classifiers.
